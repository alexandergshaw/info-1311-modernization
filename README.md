# 🚀 Modern Portfolio Template

A clean, responsive, and modern portfolio site built with vanilla HTML, CSS, and JavaScript. Designed for students to showcase their skills, projects, and personality — no frameworks required.

## ✨ Features

- **Responsive Design** — Looks great on desktop, tablet, and mobile
- **Dark / Light Mode** — Toggle with a button; respects system preference and remembers your choice
- **Smooth Scroll & Reveal Animations** — Sections animate into view as you scroll
- **Mobile Navigation** — Animated hamburger menu for small screens
- **Easy to Customize** — Plain HTML/CSS/JS — just edit and push

## 🛠 How to Customize

1. **Open `index.html`** and replace placeholder content:
   - Update "Your Name" with your real name
   - Write your own bio in the About section
   - Add your real projects with descriptions, tags, and links
   - Set your email address in the contact section
   - Update social links (GitHub, LinkedIn, Twitter, etc.)
2. **Open `styles.css`** to tweak the look:
   - Change `--color-accent` to your favorite color
   - Adjust fonts, spacing, or anything else in the `:root` variables
3. **Add a profile photo**: replace the placeholder in the About section with an `<img>` tag pointing to your photo
4. **Add project screenshots**: replace `project-image-placeholder` divs with real `<img>` tags

## 🚀 Deploy

This site is designed to work with **GitHub Pages**:

1. Push your changes to the `main` branch
2. Go to **Settings → Pages** in your repository
3. Set the source to `main` branch and `/` (root)
4. Your portfolio will be live at `https://<username>.github.io/<repo-name>/`
